# e-plantShopping - Boutique de Plantes En Ligne

Bienvenue sur le projet **e-plantShopping**, une application web moderne de commerce électronique dédiée à la vente de plantes d'intérieur et de jardin.

## Aperçu du Projet
Cette application permet aux utilisateurs de parcourir un catalogue de plantes, de filtrer les produits par catégorie (fleurs, plantes vertes, succulentes), de gérer un panier d'achat en temps réel et de simuler un processus de paiement sécurisé.

## Fonctionnalités Clés
- Catalogue de plantes dynamique avec descriptions détaillées.
- Panier d'achat interactif (Ajout, modification de quantité, suppression).
- Interface utilisateur responsive et épurée.
